题目链接：https://codeforces.com/contest/2225/problem/C

题目大意：有一个2*n的方块表格，被涂成R或B颜色，现在可以重新涂一些表格，让表格满足以下条件：

存在至少一种方式，能将所有格子化为n个配对，使得：

每队中两个格子颜色相同，并且每队中的两个格子公用一条边（即上下相邻或者左右相邻）。

题目思路：要想满足条件，即一行中的左右颜色相同或者一列中的上下颜色相同。

定义f(n)为处理完前n列所需要的最少重新涂的格子数。

状态转移：考虑最后一组怎么配对。

第i列竖配对：改成相同颜色的花费 f[i]=f[i-1]+cost_cow[i];

第i-1列和第i列横配对：第一行这两列同色+第二行这两列同色的最小花费 f[i]=f[i-2]+cost_row[i-1，i];

cost_cow:修改成这一列的代价

cost_row：修改成第一行两列相同+第二行两列相同的代价。
