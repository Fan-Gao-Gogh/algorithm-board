题目链接：https://codeforces.com/contest/2224/problem/B

题目大意：一个长度为n的非负整数数组，对其重新排序后取得(mex(a1,a2,⋯,ai)+max(a1,a2,⋯,ai))，i从1到n，取最大值的和

思路：先求max 将最大数放到第一位，即为max最大值之和；

要想mex最大，最优解一定是使数组最快达到最大mex值，即0，1，2，3.....mex。

求mex有三种情况：假设求得的mex最大值为k

1，k==0,说明数组中没有0，任意mex都为0，只输出max之和。

2，k>1且数组最大值M>k-1

3，k>1且数组最大值M==k-1

我们可以将整个数组分为两部分，第一部分为上升数组，第二部分为取得mex最大值数组

一：上升数组对应值一定为0，1，2，3...k。等差数列求和 k（k+1）/2。

如果M==k-1 那么需要在总和+1.

二：以已经取到了最大k值，剩下的数的个数为（n-（k+1））。总和为（n-（k+1））*k

那么答案为：max*n+ k（k+1）/2+（n-（k+1））*k

如果M==k-1，额外+1
