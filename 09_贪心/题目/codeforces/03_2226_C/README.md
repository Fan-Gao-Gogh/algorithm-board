题目大意：对于任意数组c，定义f(c)为执行一次操作之后，能得到的最大可能mex(c);

选择一个整数数组，使得ci=ci%bi;

给定一个长度为n的非负整数组a，求f(a)。

mex(c):表示数组 c 中最小未出现的非负整数（最小排除值）。

贪心+二分：
假设k为最小排除值

证明单调性：若k可行，则<k也可行。若k不可行，所有>=k的也不可行。

检验这个k是否为最小排除值，我们需要给0-k-1每个数分配一个a_i;

从大到小处理（multiset维护），因为小的数总能被很多大的数满足。

处理x=k-1,k-2,.....,0;

1，若x=a_i,优先使用其本身，直接移除。否则，找到>2x的最小元素（upper_bound），用它来生成x，并且移除

2，若都找不到，k则不是

证明：x mod y的数学性质：当x不变时

1，y>x 只能取到x本身

2，y<=x 只能取到<x/2的值
