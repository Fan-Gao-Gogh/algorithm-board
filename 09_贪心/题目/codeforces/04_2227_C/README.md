题目链接：https://codeforces.com/contest/2227/problem/C

题目大意：给一个包含n个正整数的数组a，定义f(a)为数组a中，乘积能被6整除的子数组的数量。

现在需要重新排列a数组，使得f(a)最小。

思路：6的质因数为2,3，因此一个子数组的乘积能被6整除，这个子数组中必须同时包含因子2和因子3。

可以将数组分为四类：

a，只有因子2

b，只有因子3

c，既有因子2也有因子3

d，既没有因子2也没有因子3

要想使f(a)最小，我们应该使a类和b类尽可能的分开，并且a/b类也应该远离c类

例如：c a d b c类会使得所有包含他的子数组都可以被6整除，而adb必须连在一起才可以被整除。
