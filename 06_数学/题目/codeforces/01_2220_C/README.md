题目大意：给定p个单位长度的直线线段以及q个为L的线段，求可以组成的最大m*n网格。

思路：假设有m行n列，设横线数量为H，竖线数量为V

1，H=n(m+1)  V=m(n+1)

假设有x个p组成横线，y个p组成竖线

2，H=q+x    V=q+y   x+y=q;

n(m+1)+m(n+1) == (q+x)+(q+y) 

等式右边代入x+y=q 右边=2*q+p

3，p+2*q ==2nm+n+m

为了将等式右边表示为两个数相乘，我们可以对等式两边乘2并且+1，得到：

2p+4q+1 == 4nm+2n+2m+1

令2p+41+1==C

4,C=(2n+1)(2m+1)

因为n和m大于等于1，所以(2n+1)和(2m+1)>=2且为奇数

令a=2n+1,b=2m+1,x>=2且y>=0

由2得 x=H-q, y=V-q,将1代入

得x=n(m+1)-q>=0  y=m(n+1)-q>=0

结合两式x-y=n-m x+y=p

得到x=p+(n-m)/2>=0  y=p-(n-m)>=0

得出|n-m|<=p

由b-a == 2(m-n)<=2p可得

b-a<=2p

总体思路：

1，先找sqrt（C）,找到他的两个>=3并且为奇数的因数

2，判断b-a是否<=2p

3，n=(a-1)/2  m=(b-1)/2
